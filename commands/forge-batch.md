# /forge-batch

여러 챕터의 대본을 일괄 생성한다.

## 사전 조건

- `output/_series_guide.json`이 **반드시** 있어야 한다 (장르/스타일 통일)
- 시리즈 가이드가 없으면 먼저 StoryLens의 /consult 실행을 안내한다
- 최소 1개 챕터를 /forge로 개별 생성하여 품질 확인 후 사용을 권장

## 실행 흐름

### Step 1: 대상 챕터 확인

사용자 지정에 따라 대상 결정:
- `all` → sample/ 폴더의 모든 챕터 (이미 생성된 챕터는 건너뜀)
- `1-5` → 1장~5장
- `3,7,12` → 지정 챕터만
- 이미 output/에 있는 챕터는 표시 (덮어쓸지 확인)

**대상 목록 + 시리즈 가이드 요약을 보여주고 확인 대기.**

### Step 2: 챕터별 순차 처리

각 챕터에 대해:

1. **분석 + 라인업 제안** (analyze-chapter)
   - 씬 라인업을 1줄 요약으로 표시 (상세 X)
   ```
   📋 3장: 5씬 — 하버드교실 → 스키너분노 → 설계도 → 프레시 → 자동교사
   ```

2. **자동 프롬프트 생성** (prompt-craft)
   - 상세를 보여주지 않고 자동 생성
   - 문제가 있으면 자동으로 조정

3. **파일 저장** (generate-script)
   - JSON + MD + CSV 저장
   - 매니페스트 업데이트

4. **1줄 완료 보고**
   ```
   ✅ 3장 완료 (5씬, 2분 0초) → output/ch03/
   ```

### Step 3: 중간 확인

5개 챕터마다 한 번 멈추고 사용자에게 중간 확인:
```
📊 진행 상황: 5/17 완료
   1장 ✅ (6씬, 2:30)
   2장 ✅ (7씬, 3:00)
   3장 ✅ (5씬, 2:00)
   4장 ✅ (6씬, 2:45)
   5장 ✅ (5씬, 2:15)

계속 진행할까요? (한 챕터를 열어서 확인하고 싶으시면 말씀해주세요)
```

### Step 4: 최종 보고

```
✅ 일괄 생성 완료

📊 전체 요약:
   처리: 17개 챕터
   총 씬: 102개
   총 시간: 42분 30초
   이미지 수: 102장

📁 산출물:
   output/ch01/ ~ output/ch17/
   output/_manifest.json (업데이트됨)

💡 다음 단계:
   - output/ 폴더의 JSON을 FlowGenie에서 배치 처리
   - 개별 챕터 수정: /forge-review [챕터번호]
```

## 배치 모드 vs 개별 모드 차이

| 항목 | /forge (개별) | /forge-batch (일괄) |
|------|--------------|-------------------|
| 씬 라인업 확인 | 상세 + 수정 가능 | 1줄 요약만 |
| 프롬프트 확인 | 2-3씬씩 + 수정 가능 | 자동 생성 (확인 안 함) |
| 사용자 개입 | 매 단계 | 5챕터마다 중간 확인 |
| 품질 | 높음 (수동 검수) | 보통 (자동, 나중에 review) |
| 속도 | 느림 | 빠름 |
